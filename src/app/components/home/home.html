Welcome to the ooniprobe web interface

Through this web interface you will be able to start ooniprobe measurements and view their results.

Here are some things that you should know:

  • There is a daemon running on your computer that is responsible for hosting this web interface and triggering periodic tasks
  • Periodic tasks include:
    • Deleting measurements that are older than 7 days
    • Running the decks that have been configured on this system
    • Updating the inputs used by ooniprobe (every day)

To get started with running some measurements click on the nettest button in the upper right corner
