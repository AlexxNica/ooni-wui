Welcome to the ooniprobe web interface!

Through this web interface you will be able to run ooniprobe tests to examine censorship in your network and to view the results.

To get started, here are some things you should know:

Running tests automatically

Click on the decks button to configure ooniprobe to automatically test your network

Running single tests

Click on the nettests button to choose the ooniprobe tests that you want to run

Viewing results

Click on the measurements button to view the results of the tests that you ran

Happy testing!
